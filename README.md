# 📋 Task Board — Kanban приложение

Интерактивная доска задач (Kanban Board), созданная на **HTML, CSS и JavaScript**.
Позволяет удобно управлять задачами: добавлять, редактировать, перемещать между колонками и хранить данные в браузере.

---

## 🚀 Возможности

✅ Добавление задач
✅ Удаление задач
✅ Редактирование (двойной клик)
✅ Drag & Drop между колонками
✅ Поиск задач 🔍
✅ Сохранение в LocalStorage
✅ Темная тема 🌙
✅ Адаптив под мобильные устройства 📱
✅ Анимации появления ✨

## 🎯 Как пользоваться

* ➕ Введи задачу → нажми **"Добавить"**
* 🔍 Используй поиск для фильтрации задач
* ✏️ Двойной клик по задаче — редактирование
* ❌ Кнопка "✖" — удалить задачу
* 🖱 Перетаскивай задачи между колонками
* 🌙 Кнопка сверху — переключение темы

---

## 🧠 Как работает

* Все задачи хранятся в массиве `tasks`
* Данные сохраняются в **localStorage**
* Каждая задача имеет:


{
  id: number,
  text: string,
  status: "start" | "progress" | "done"
}


* Интерфейс обновляется через функцию `render()`
